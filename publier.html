    Chatbot sur mon site web — les étapes

    Même IA que le SMS. Une bulle de chat apparaît en bas à droite de votre site.

    1. Configurez votre agent dans Chatbot (services, FAQ, ton).
    2. Dans Paramètres, cochez Activer le widget web et enregistrez.
    3. Copiez le code ci-dessous.
    4. Collez-le juste avant </body> sur votre site (ou dans la section « code personnalisé / footer »).
    5. Publiez / mettez en ligne la page, puis rechargez votre site — la bulle 💬 doit apparaître.
    6. Testez : envoyez une question (prix, horaires). La conversation arrive aussi dans Conversations.

    Où coller selon votre plateforme

    • WordPress — Apparence → Éditeur de thème / Pied de page, ou plugin « Insert Headers and Footers »
    • Wix — Paramètres → Code personnalisé → Body - end
    • Squarespace — Paramètres → Code d'injection avancé → Footer
    • Shopify — Boutique en ligne → Thèmes → Modifier le code → theme.liquid avant </body>
    • Site HTML / Webflow — avant la balise de fermeture </body>
    —

    Rien ne s'affiche? Vérifiez que le widget est activé, que le code est bien publié, et faites un hard refresh (Ctrl+Shift+R).
