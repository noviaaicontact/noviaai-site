Étapes

    Agent sur mon site web — les étapes

    Même IA que le SMS. Une bulle 💬 apparaît en bas à droite de votre site.

    1. Préparer NoviaAI

    1. Configurez services et FAQ dans Agent.
    2. Dans Paramètres → cochez Activer le widget web → Enregistrer.

    2. Copier votre code

    Cliquez Copier. Vous le collerez à l’étape 3 (ne le modifiez pas).

    —

    3. Coller le code sur votre site (l’étape importante)

    Choisissez comment votre site est fait — on vous dit exactement où cliquer :

    1. Ouvrez Wix → votre site → Tableau de bord (pas l’éditeur de design).
    2. Menu de gauche → Paramètres.
    3. Descendez jusqu’à Code personnalisé (parfois sous « Avancé ») → ouvrez-le.
    4. Cliquez + Ajouter un code personnalisé.
    5. Collez le code NoviaAI (Ctrl+V / Cmd+V) dans la zone de texte.
    6. Nom : ex. NoviaAI agent.
    7. Emplacement : Body - end (fin du body) — pas Header.
    8. Pages : Toutes les pages.
    9. Cliquez Appliquer, puis Publier votre site Wix.
    1. Connectez-vous à l’admin WordPress (/wp-admin).
    2. Installez le plugin gratuit « WPCode » ou « Insert Headers and Footers » (Plugins → Ajouter).
    3. Ouvrez le plugin → section Footer / Body (pas Header).
    4. Collez le code NoviaAI → Enregistrer.
    5. Rechargez votre site public (pas l’admin) — la bulle doit apparaître.

    Sans plugin : Apparence → Éditeur de thème → fichier footer.php → collez juste avant </body> (réservé si vous êtes à l’aise).

    1. Squarespace → Paramètres → Advanced / Avancé.
    2. Ouvrez Code Injection / Injection de code.
    3. Dans la case FOOTER, collez le code NoviaAI.
    4. Enregistrez, puis Save / publiez si demandé.
    1. Admin Shopify → Boutique en ligne → Thèmes.
    2. Sur le thème actuel → ⋯ → Modifier le code.
    3. À gauche, ouvrez le fichier theme.liquid (souvent sous Layout).
    4. Descendez tout en bas. Cherchez la ligne </body>.
    5. Collez le code NoviaAI juste au-dessus de cette ligne.
    6. Cliquez Enregistrer.
    1. Webflow → Project settings (icône engrenage) → Custom Code.
    2. Section Footer Code → collez le code NoviaAI.
    3. Save → Publish le site.
    1. Ouvrez le fichier de votre page (souvent index.html) dans un éditeur.
    2. Allez tout en bas du fichier.
    3. Trouvez </body> — c’est la fin de la page.
    4. Collez le code NoviaAI sur la ligne juste au-dessus de </body>.
    5. Enregistrez le fichier et remettez-le en ligne (FTP, hébergeur, etc.).

    Pas besoin de tout comprendre. Envoyez ceci à la personne qui gère votre site :

    « Peux-tu coller ce script NoviaAI juste avant </body> (ou dans le code footer / Body end) sur toutes les pages, puis republier le site? »

    Joignez le code copié à l’étape 2. Ensuite testez vous-même à l’étape 4.

    4. Vérifier que ça marche

    1. Ouvrez votre site public (pas l’éditeur) → hard refresh : Ctrl+Shift+R.
    2. Vous devez voir une bulle en bas à droite → cliquez → posez une question (ex. « Vos horaires? »).
    3. La conversation apparaît aussi dans Conversations.

    Toujours rien? Widget activé + enregistré? Code collé dans Body / Footer (pas Header)? Site bien publié?
