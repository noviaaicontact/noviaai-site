Questions fréquentes

Les réponses en 30 secondes.

Est-ce que je dois changer mon numéro?

Non. Option A : on vous donne une ligne NoviaAI à publier sur Google. Option B : vous gardez votre numéro et activez un renvoi d'appels manqués (guide Bell, Vidéotron, Rogers, TELUS… dans Publier).

Et si je réponds à l'appel?

Votre cellulaire sonne en premier. NoviaAI n'envoie le SMS et ne prend le relais que si vous ne répondez pas.

Est-ce que je peux répondre moi-même aux textos?

Oui. Toutes les conversations sont dans votre inbox. L'IA peut démarrer ; vous reprenez quand vous voulez, depuis le tableau de bord.

Comment fonctionne l'essai 14 jours?

Vous ajoutez une carte pour activer la ligne. Pendant 14 jours, pas de prélèvement. Ensuite : 199 $ CAD / mois. Vous annulez en un clic avant la fin de l'essai si ce n'est pas pour vous.

Comment mettre le chatbot sur mon site?

Activez le widget dans Paramètres, copiez le code, puis suivez le guide étape 3 dans Publier : choisissez Wix, WordPress, Shopify, etc. — on indique où cliquer pour coller le code. Republiez votre site : une bulle de chat apparaît (même IA que le SMS).

L'IA réserve-t-elle dans mon agenda?

Non. Elle peut envoyer votre lien de réservation (Calendly, etc.), noter une demande et vous alerter. C'est vous qui confirmez le rendez-vous.
